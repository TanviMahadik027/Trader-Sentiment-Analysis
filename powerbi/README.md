# Power BI Dashboard

## Trader Performance vs Market Sentiment Analysis

This Power BI dashboard provides an interactive analysis of cryptocurrency trader performance and its relationship with market sentiment.

## Key Metrics

- Total Trades: 211K
- Total PnL: 10.30M
- Average PnL: 48.75
- Buy Trades: 103K
- Sell Trades: 109K
- Win Rate: 41.1%

## Dashboard Analysis

The dashboard includes:

- Trader performance KPIs
- Market sentiment analysis
- PnL by market sentiment
- Average PnL by year
- Buy vs Sell distribution
- PnL by cryptocurrency
- Interactive filters for Market Sentiment and Coin

## Conditional Formatting

- Win Rate above 50% → Green
- Win Rate between 40% and 50% → Orange
- Win Rate below 40% → Red
- Positive PnL → Green
- Negative PnL → Red
